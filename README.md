# Basic MERN Blog

# Setup
Install server and dev dependencies

## Dependencies
- [Brew](https://brew.sh/)

`/usr/bin/ruby -e "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)"`
- Yarn

`brew install yarn`

- Nodemon (set globally)
`npm install -g nodemon`

Install dependencies

`yarn up`

Run application

`yarn dev`
